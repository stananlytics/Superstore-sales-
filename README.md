Project Overview
This project presents an interactive Superstore Customer Sales Dashboard designed to analyze sales and profitability across regions, categories, and states.
The dashboard enables stakeholders to quickly identify performance trends, profit drivers, and loss-making areas using key performance indicators (KPIs) and visual analytics.
